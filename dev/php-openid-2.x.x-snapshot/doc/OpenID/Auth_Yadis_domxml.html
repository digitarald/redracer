Class Auth_Yadis_domxml

Description
Description | Methods (details)

This concrete implementation of Auth_Yadis_XMLParser implements

the appropriate API for the 'domxml' extension which is typically packaged with PHP 4. This class will be used whenever the 'domxml' extension is detected. See the Auth_Yadis_XMLParser class for details on this class's methods.
